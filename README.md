Обработчик файлов 2.0 

Сборка:
- Открыть CMakeLists.txt в Qt Creator
- Выбрать комплект MinGW
- Нажать "Собрать"

Функциональность:
- пользовательский выбор входной и выходной директорий
- режимы: единичный или по таймеру
- модификация файлов XOR с 8-байтным ключом
- буферизованное чтение 
- многопоточная обработка (реализована при помощи пула потоков)
- максимальное количество потоков - половина от количества логических ядер
- каждый поток имеет свой буфер размером 2 МБ
- возможность мгновенной остановки без повреждения файлов
- доступ к общим данным синхронизирован при помощи std::atomic

Cтруктура:
- fileModifier - главное окно
- fileProcessor - обработчик (управляет пулом потоков и обработкой)
- fileProcess - задача для обработки одного файла (наследник QRunnable)


В коде для удобства оставлены отладочные сообщения qDebug()
